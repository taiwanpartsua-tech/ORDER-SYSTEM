## Налаштування для Netlify

**ВАЖЛИВО:** Якщо на production сайт показує білий екран або помилку "Failed to fetch", потрібно налаштувати змінні середовища. Детальні інструкції в [NETLIFY_SETUP.md](./NETLIFY_SETUP.md).
